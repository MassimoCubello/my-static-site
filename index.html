The Building Blocks of the Web

Discover how HTML, CSS, and JavaScript work together to createweb pages!

What is HTML?

HTML (HyperText Markup Language) is a vital language for creating web pages. It provides the basic structure of any webpage (think of the skeleton's role in the human body), which is later enhanced and modified by other languages like CSS and JavaScript.

HTML allows you to:

  • Add headings, paragraphs, and links
  • Organize content into sections and lists
  • Create forms for users to input data

What is CSS?

CSS (Cascading Style Sheets) is the language used to make websites look a certain way, adding visual flair to the finished product and surrounds the copy that's been written in the HTML file.

CSS allows you to:

  • Design the layout of a webpage
  • Change colors and fonts
  • Utilize flexbox to organize content

What is JavaScript?

JavaScript is the language that allows users to interact with a website. It lets developers build logic and systems into any given web page, making them respond to user input.

JavaScript allows you to:

  • Create unique elements and rulesets
  • Complement HTML and CSS files
  • Build complex web applications
